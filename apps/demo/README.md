# Gradio Demo

A minimal web UI for manual testing of the phishing email detection agent.

## Try it locally

```
python apps/demo/gradio_app.py
```

## Example input

Paste a raw email (.eml) body in the text box, e.g.:

```
From: "IT Support" <it-support@example.com>
To: user@example.com
Subject: Urgent account verification

Please verify your account immediately by visiting https://example.com/verify
```
